APART

Apart is a co-op puzzle game I made for the 2021 Brackey's Game Jam with the theme: Stronger Together. Your goal is to get both scientists to the end of each level and escape the locked-down facility. It was made in the Unity Game Engine in one week.

https://npilon03.itch.io/apart

SKAL

Skal is a compilation of micro games similar to the Wario Ware series by Inteligent Systems and Nintendo. Your goal is to get as many drinks as possible by completing mini games themed (mostly) around vikings. It was my entry for My First Game Jam: 2021. This game was developed in Unity C# in two weeks.

https://npilon03.itch.io/skal

SDL 2.0 Frogger Game

This game was made using SDL 2.0 and takes inspiration from Frogger. Your goal is to deliever packages to the other side of the factory while avoiding carts and crossing pools of acid. I learned a lot about memory management and object oriented programming while developing this game.

SDL 2.0 Monkey Runner

This game was also made using SDL 2.0 and takes inspiration from Jetpack Joyride by Halfbrick Studios. It is an endless runner where you collect bananas and dodge spears. I learned a lot about controller inputs and more memory management with this project.

OpenXR Demo Project

This is a VR Pokemon styled demo I made with a collegue for our final graphics programming project. There isn't much in terms of gameplay but you can throw balls at creatures to add them to your collection. You can keep track of creatures you have collected and spawn balls from your left hand UI. We built this using Unity and OpenXR. We also used Blender for the models.

You can find the project files here: https://github.com/NolanPilon/VRGraphicsProject
